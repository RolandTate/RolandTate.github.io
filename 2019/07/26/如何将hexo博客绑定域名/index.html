发表于 2019-07-26 分类于 hexo

#

如何将hexo博客绑定域名
如果是利用hexo搭建博客并托管在github，可以通过购买域名后，将博客地址和域名绑定，这样就可以通过访问购买的域名来访问博客了。

##购买域名
我是通过阿里云来购买的域名，可以通过淘宝登陆并完成付款，在完成相关信息的填写后购买一个心仪的域名即可。

##域名解析
购买完域名后，在域名控制台里对购买的域名点击解析，然后添加记录，记录类型选择A, 总共添加两条，第一条主机记录为www，记录值为192.30.252.154。第二条的主机记录不填，记录值为192.30.252.153，线路都是默认即可。这两个地址都是Github提供的。

##修改Github相关文件

  1. ###创建CNAME文件
    在托管hexo的仓库根目录下新建一个文件名为CNAME，内容填写你购买的域名，不用加https://或www.之类的东西，只用填写购买的域名。
  2. ###在GitHub Pages添加域名
    在托管的仓库中找到setting,往下拉，找到GitHub pages,然后在Custom domain填入购买的域名，和CNAME内容一样，然后save。

##修改站点配置文件
将站点地址更新为绑定的域名，在_config.yml文件中找到

# URL
## If your site is put in a subdirectory, set url as 'http://yoursite.com/child' and root as '/child/'
url:

在url后面填写完整的域名http://www.你购买的域名，注意冒号后有一个空格。
最后使用命令hexo clean,hexo g,hexo d完成部署即可。
一般配置完成后不能立即访问，需等待，可能需要一天，祝你好运。

Head First Java学习笔记

发表于 2019-08-31 更新于 2019-09-01 分类于 Java

引用

对象引用

  • 对象引用变量保存的是存取对象的方法，它不是对象的容器，而是类似指向对象的指针。或者可以说是地址，但在Java中我们不会也不该知道引用变量中实际装载的是什么，它只是用来代表单一的对象。只有Java虚拟机才会知道如何使用引用来取得对象。
  • primitive主数据类型变量是以字节来代表实际的变量值，但对象引用变量却是以字节来表示取得对象的方法。
  • 数组也是对象，一旦数组被声明出来，你就只能装入所声明类型的元素。

认识Java的API

ArrayList

  1. 创建
    ArrayList<Egg> myList=new ArrayList<Egg>();
  2. 加入元素
    Egg s=new Egg(); myList.add(s);
  3. 查询大小
    int theSize=myList.size();
    因为myList有一个元素，所以size()返回1
  4. 查询特定元素
    boolean isIn=myList.contains(s);
    因为myList带有s所引用的Egg对象，所以此方法会返回true
  5. 查询特定元素的位置
    int idx=myList.indexOf(s);
    ArrayList是零基的，所以s引用的对象是第一个对象，而indexOf()会返回0
  6. 判断集合是否为空
    boolean empty=myList.isEmpty();
    因为不是空的，isEmpty()会返回false
  7. 删除元素
    myList.remove(s);

使用函数库(Java API)

在Java的API中，类是被包装在包中。要使用API中的类，你必须知道它被放在哪个包中。

使用类的方法

  • import
    放一个import述句在程序源文件的最前面:
    import java.util.ArrayList; public class MyClass{...}
  • type
    或者在程序代码中打出全名。不管在哪里，只要有使用到就打出全名。除非是来自java.lang这个包中。

如何查询API

  • 查阅参考书
  • 查阅HTML API文

继承与多态

了解继承

  • 继承的关系意味着子类继承了父类的方法。当我们提及“类的成员时”，成员的意思就是实例变量和方法。

  • 用关键字extends来继承。

  • 继承时子类会继承父类的变量和方法，子类可以覆盖同名的父类的方法，但无法覆盖变量，可以通过super来访问父类的方法和变量。覆盖方法时不能降低方法的存取权限且参数必须要一样且返回类型必须要兼容（声明一样的类型或该类型的子类）。

  • 当调用对象引用的方法时，会调用到与该对象类型最接近的方法，也可以说是层次树的最下方的方法。

多态

  • 在多态下，引用与对象可以是不同的类型，引用类型可以是实际对象类型的父类
    Animal myDog=new Dog();
  • 参数与返回类型也可以多态。将参数与返回类型声明为父类的类型，那么任何继承该父类的子类都可以传入、返回

接口与抽象类

抽象类

  • 有些类不该被实例化，但又需要它来继承和产生多态时，使用关键字abstract来将该类变为抽象类，如
    abstract class Canie extends Animal{...}
  • 抽象类不能创建实例，但还是可以用抽象类作为引用类型来实现多态
  • 除了类之外，方法也可以是abstract的，抽象的类代表此类必须要被子类实现，抽象的方法表示此方法必须要被子类覆盖。
    public abstract void eat();
    只有抽象类才可以拥有抽象的方法。

Object

Object这个类是所有类的源头;他是所有类的父类。

  • 不要滥用Object类型，例如:
    当你把对象装进ArrayList<Dog>时，它会被当作Dog来输入与输出，但当你把它声明成ArrayList<Object>时，任何从 ArrayList<Object>中取出的东西就都会被当作Object类型的引用，除非你自己进行类型转换。

接口

Java不支持多重继承，因为多重继承会有成为“致命方块”的问题。如果要解决多重继承的问题可以使用接口。接口解决致命方块的办法很简单：把全部的方法设为抽象的。

接口的定义与实现

接口的定义

public interface Pet{...}

接口的实现

public class Dog extends Canine implements Pet{...}

  • 使用implements这个关键字来继承接口
  • 接口的方法带有public和abstract的意义，这两个修饰符是属于选择性的(可以不打出来)
  • 类可以实现多个接口
  • 任何接口的子类都得实现接口的方法

构造器

栈与堆：生存空间

  • 实例变量是声明在类中方法之外的地方。局部变量声明在方法或方法的参数上。
  • 所有局部变量都存在于栈上相对应的堆栈块中。
  • 对象引用变量与primitive主数据类型变量都是放在栈上。
  • 不管是实例变量或局部变量，对象本身都会在堆上。

构造函数

构造函数用来初始化对象的状态。使用方法和C++一样。

父类的构造函数在对象的生命中所扮演的角色

  • 在创建新对象时，所有继承下来的构造函数都会执行。
  • 构造函数在执行时，第一件事是去执行它的父类的构造函数，这会连锁反应到Object这个类为止

如何调用父类的构造函数

  • 调用父类构造函数的唯一方法是调用super()，且必须是第一行调用
  • 如果我们没有调用super()，编译器会帮我们加上super()的调用(如果有多个重载版本的构造函数，编译器加上的还是无参数版本)

从某个构造函数调用重载版的另一个构造函数

  • 在构造函数中使用this()，且将this()放在第一行
  • super()与this()只能有一个

对象的存活周期

对象的声明周期要看引用的生命周期而定。如果对象的最后一个引用消失，对象会被GC回收

数字与静态

静态方法

用static关键字修饰方法，静态方法与实例无关，调用时，以类的名称来调用。
带有静态的方法通常(虽然不一定是这样)不打算初始化，你可以用私有的构造函数来限制非抽象类被初始化，被标记为private的方法代表只能被同一类程序调用，所以将构造函数标记为私有，编译器就会知道不能创建这个类的实例。

静态的方法不能调用非静态的变量

静态的方法是在无关特定类的实例情况下执行的。因为静态的方法时通过类的名称来调用，所以静态的方法无法引用到该类的任何实例变量。如果尝试在静态的方法内使用实例变量，编译器会报错。

静态的方法也不能调用非静态的方法

非静态的方法通常时以实例变量的状态来影响该方法的行为，哪怕非静态方法没有使用到实例变量，也不会通过编译。

静态变量

它的值对所有实例来说都相同，同一类所有的实例变量共享一份静态变量。

静态变量的初始化

静态变量是在类被加载时初始化的且会在该类的任何静态方法执行之前就初始化。如果你没有给静态变量赋初值，他就会被设置为默认值。

静态的final变量是常数

一个被标记为fianl的变量代表它一旦被初始化之后就不会改动。也就是说类加载之后静态fianl变量就一直会维持原值。
常数变量的名称应该要都是大写字母。

静态final变量的初始化

  • 声明的时候
    public class Foo{public static final int FOO_X=25;}

  • 在静态初始化程序中

静态初始化程序是一段在加载类时会执行的程序代码，它会在其他程序可以使用该类之前就执行。

final

  • final的变量代表你不能改变他的值。
  • final的方法代表你不能覆盖该方法
  • final的类代表你不能继承该类

数字

数字的格式化

在Java中，数字与日期的格式化功能并没有结合在输出/输入功能上。通常对用户显示数字时通过GUI来进行的。你会把String放在可滚动的文字区域块或表格中。从Java 5.0起，更多更好更有扩展性的功能时通过java.util中的Formatter这个类来提供的。我们可以通过调用静态的String.format()并传入值与格式设定就可以实现数字的格式化。

将数字以带逗号的形式格式化

输出结果

1,000,000,000

解析格式化结构

基本上来说，格式化由两个主要部分组成(不只是这样)：

  1. 格式指令
  2. 要格式化的值

format()方法的第一个参数被称为“格式化串”，它可以带有实际上就是要这么输出而不用转译的字符。当你看到%符号时，要把它想做是会被方法其余参数替换掉的位置。
例如
format("I have %.2f bugs to fix.",476578.09876);
输出

I have 476578.10 bugs to fix.

此例的%符号是第二个参数会放置的位置，“.2f”代表该参数要使用的格式，其他的字都会以原来的方式输出。
如果加上逗号
format("I have %,.2f bugs to fix.",476578.09876);
输出

I have 476,578.10 bugs to fix.

“格式化说明”的格式

跟在百分号后面包括类型指示（像是d或f）的每个东西都是格式化指令，除非遇到新的百分号，在类型指示之后的一组字符，格式化程序会假设都是直接输出的字符串
格式化说明最多会有5个部分（不包括%符号）。下面的[]符号里都是选择性的项目，因此只有%与type是必要的。格式化说明的顺序是有规定的，必须要以这个顺序来指定。
%[argument number][flags][width][.precision]type

argument number:如果要格式化的参数超过一个以上，可以在这里指定是哪一个。
flags:特定类型的特定选项，例如数字要加逗号或正负号
width:最小的字符数，注意：这不是总数；输出可以超过此宽度，若不足则会主动补零
.precision:精确度，注意前面有个圆点符号
type:一定要指定的类型标识

日期

如果你要输出这样的字符串：Sunday,Nov 28 2004
看起来好像没什么特别。但是如果说是要把Date类型的变量日期用这样的格式输出呢？Date类型是Java上表示时间用的。
数值与日期时间格式化的主要差别在于日期格式的类型使用“t”开头的两个字符表示，下面有几个范例：
完整的日期与时间：%tc
String.format("%tc",new Date());

Sun Nov 28 14:52:41 MST 2004

只有时间：%tr
String.format("%tr",new Date());

03:01:47 PM

周、月、日：%tA%tB%td
因为没有刚好符合我们要求的输出格式，所以得组合三种形式来产生出所需要的格式：

Date today=new Date(); String.format("%tA,%tB %td",today,today,today);

Sunday,November 28

同上，但不用重复给参数
Date today=new Date(); String.format("%tA,%<tB %<td",today");
